基本操作

  • 如上引用：大于号+空格
  • 有序无序列表
  • 减号+空格就是一个无序列表
  1. 有序列表，数字+英文句号+空格

  • 百度超链接跳转,方括号+圆括号。方括号是跳转链接，圆括号是跳转的网址
    百度

  • 插入图片：感叹号+中括号+小括号。中括号中是图片的描述实际不显示，小括号中是图片地址。

  • 文字加粗：在要加粗文字外加两个星号，shift+8。

  • 斜体：文字外面加一个星号。
